# CTWeb
Clevertap Web Sample

Open project in VS Code & install an extension called as Preview on Web Server (by YuichiNukiyama)
Once that is done open index.html file in vs code and right click.
You will see an option below as follows: vs-code-preview-server

<img width="378" alt="image" src="https://user-images.githubusercontent.com/16386061/209905480-4a0fc618-af96-4ae0-9564-ba9ae7b75e80.png">

This will open your website in a localhost environment in your browser.
To show the notification permission popup click on the send notification button.
